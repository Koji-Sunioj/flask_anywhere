The Data

This data represented here is based on the W3 schools SQL tutorial, which is a fictional company with customers, suppliers and several other details common in a business.

The use case for this interface is a sales associate or decision maker in the company wanting to visualize the performance, in which case all values and categories are centered around the client's order details.

There are two different plot types available: correlative and aggregate.

Correlative simply plots one sales value or category vs. another, and is more of a utility to view where data is clustered, and by category (if one is chosen).

Aggregate plotting is more refined, offering clear insights of plotted data with typical line, column plots where aggregate types (sum, average) and time series can by filtered.

Some sales categories have a lot of data points and work is being done to simplify the visualizations to make it more readable.
